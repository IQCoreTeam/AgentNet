# AgentNet - 디자인 시스템 & 컨셉 (Design Language)

> 정체: 실제 UI가 따라야 할 하드 룰(아이콘/카피/토큰)과 추상 디자인 컨셉(느낌)을 적어두는 문서.
> 토큰 "값"의 최종 확정은 뒤의 시각 딥리서치 단계에서 레퍼런스 앱 보고 잠근다. 지금은 규칙 + 초안값 + 컨셉.
> 코드 기준: 색 토큰은 이미 `surfaces/webview/src/index.css`에 `--an-*`로 존재. 날짜: 2026-06-23.

---

## 1. 하드 룰 (UI 제약, 예외 없음)

1. **실제 화면에 이모지 금지.** 아이콘이 필요하면 이모지 대신 **SVG**로 그려 넣는다. (본 문서·와이어프레임의 그림문자는 자리표시일 뿐, 구현에는 안 들어간다.)
2. **em-dash(—)·en-dash(–) 금지** (UI 카피). 쉼표·콜론·괄호·일반 하이픈으로 처리.
3. **폰트 크기는 중앙화 토큰 몇 개만.** 임의 `text-[0.xx rem]` 금지. 정해진 스케일(§3) 안에서만.
4. **색은 중앙화 토큰만.** 생 hex 금지, `--an-*`(§4)만 사용. 현재 코드에 흩어진 생 hex(`#00E673` 네온 등)는 토큰으로 수렴시킨다.
5. **다크 모드 우선.** 기본은 다크. 라이트 모드는 후순위(필요 시 같은 토큰의 라이트 매핑으로).
6. 최소 터치 타깃 44px(`--an-tap`), 둥근 모서리는 정해진 radius 토큰만.

---

## 2. 디자인 컨셉 (추상 취향, 느낌 잡기)

한 줄: **"미니멀한 다크 코더 툴인데, IQ만의 은근한 해커 감성이 있고, 내가 가진 게(스킬/성취) 기분 좋게 드러나는."**

- **다크 중심 / 미니멀 코더:** near-black 베이스에 레이어드 surface, hairline 보더, 여백 넉넉. 터미널/브랜치(개발도구) 정서. 장식 최소, 구조로 말한다. 채팅이 코어라 ChatGPT/Claude의 차분함은 유지.
- **IQ만의 해커 감성, 단 매니악하지 않게:** 절제된 그린 액센트(`--an-green`은 의도적으로 "refined, not neon") + 모노스페이스 디테일(코드/주소/세션 ID)로 해커 무드. 단 matrix 코스프레/네온 범벅은 피하고, 비크립토 바이브코더도 편하게 다가오는 선에서.
- **듀오링고식 "보유 자랑":** 스킬을 사거나 성취가 쌓이면 그게 **기분 좋게 보이게.** 소유/희귀에 은근한 글로우, 컬렉션이 잘 진열되는 느낌, 작은 축하 모먼트(이미 BuyCelebration 존재). 단 가짜 희소성/다크패턴 없이 정직한 신호로만(자세한 가드레일은 game-plan.md).
- **모션:** 목적 있는 미세 모션만(떠다니는 바 스크롤 자동숨김, 구매 축하 등). 과하지 않게.

레퍼런스(시각 딥리서치 때 정독): Linear / Raycast / Vercel(다크 개발 미니멀), Things(차분), Duolingo(보유·성취 딜라이트), Phantom(모던 크립토), ChatGPT·Claude(챗 친숙), IDE 다크 테마(해커 무드).

## 3. 타입 스케일 (초안, 모바일 기준, 6단)

> 현재 코드엔 폰트 토큰이 없고 임의값 9종(0.62, 0.68, 0.72, 0.95, 1.02, 1.12, 1.18rem) + `text-xs/sm/lg`가 난립.
> 이를 **6단**으로 수렴. 값은 iOS HIG / Material / 모바일 챗앱 관행 기준 초안(딥리서치에서 잠금).

| 토큰 | px / rem | 용도 |
|---|---|---|
| `text-micro` | 11 / 0.6875 | 탭 라벨, 뱃지, 캡션 |
| `text-small` | 13 / 0.8125 | 보조 텍스트, 메타, 타임스탬프 |
| `text-body` | 15 / 0.9375 | 기본 본문, 메시지, 입력값 |
| `text-title` | 17 / 1.0625 | 행 타이틀, 카드 제목 |
| `text-heading` | 22 / 1.375 | 화면 제목 |
| `text-display` | 28 / 1.75 | 온보딩 히어로 |

- **웨이트 3종만:** regular 400 / medium 500 / semibold 600.
- **라인하이트 2종:** tight 1.2(제목), normal 1.45(본문).
- 매핑 가이드: 기존 0.62~0.72 to micro/small, 0.95 to body, 1.02~1.18 to title, `text-xs` to small, `text-sm` to body.

## 4. 컬러 토큰 (다크 우선, 코드의 `--an-*` 정본)

> 색은 이미 `index.css :root`에 정의됨. 이걸 **정본**으로 두고, 흩어진 생 hex를 여기로 수렴.

**Surface (레이어드 다크)**
| 토큰 | 값 | 용도 |
|---|---|---|
| `--an-bg-0` | #0a0b0d | 최하층(페이지) |
| `--an-bg-1` | #141519 | 상승면(헤더, 카드, 드로어) |
| `--an-bg-2` | #1c1e23 | 최상층(입력, 활성 행, 버블) |
| `--an-bg-glass` | rgba(16,17,21,.82) | 떠다니는 바/블러 |

**Text**
| `--an-fg` #ececee | 본문 | `--an-fg-dim` #9a9aa3 | 보조 | `--an-fg-mute` #66666e | 3차/placeholder |

**Line:** `--an-line` rgba(255,255,255,.085) / `--an-line-soft` .05 (hairline).

**Accent / Semantic**
| 토큰 | 값 | 의미 |
|---|---|---|
| `--an-green` (+soft/line/dim) | #3ac07a | 브랜드 액센트(refined, not neon) |
| `--an-amber` | #e0a23a | 경고/compaction |
| `--an-red` (신규로 토큰화) | #e5484d | 위험/삭제 (현재 생 hex) |
| `--an-violet` | #a98bff | 스킬 발행 액센트 |

**Radius:** `--an-radius` 16 / `--an-radius-sm` 11 / `--an-radius-pill` 999. **Tap:** `--an-tap` 44.

**정리 작업(부채):** `#00E673`(네온 그린) 23곳, `#00d068`, 기타 생 hex to 위 토큰으로 치환.

## 5. 아이콘

- **SVG only**(이모지 금지). stroke 기반 라인 아이콘, 1.5px 안팎 일관 두께.
- 사이즈 토큰화 권장(예: 18 / 22 / 24). 색은 `currentColor`로 받아 텍스트 토큰 상속.
